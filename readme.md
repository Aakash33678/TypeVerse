
# ⌨️ TypeVerse

A modern real-time multiplayer typing platform where players can compete with friends, create private rooms, and improve their typing speed through live races.

> Built with **Next.js**, **Socket.IO**, **PostgreSQL**, **Drizzle ORM**, and **Better Auth**.

---

## 🚀 Live Demo

**Website:** https://type-verse-phi.vercel.app/

---

## ✨ Features

### 🔐 Authentication
- Secure user authentication
- Persistent login sessions
- Protected routes

### 👥 Friends System
- Send and receive friend requests
- Accept or reject requests
- View online/offline status
- Friends list management

### 🏁 Multiplayer Racing
- Create private multiplayer rooms
- Join existing rooms
- Live synchronized typing races
- Real-time player progress updates
- Race countdown
- Race completion tracking

### 📊 Performance Tracking
- Words Per Minute (WPM)
- Accuracy calculation
- Race statistics
- Personal performance tracking

### ⚡ Real-Time Communication
- Socket.IO powered multiplayer
- Instant room synchronization
- Live player status
- Low-latency gameplay

### 🎨 Modern UI
- Responsive design
- Dark mode interface
- Smooth animations
- Mobile-friendly layout
